# Делаю тутор на гит 

## первые команды гита 

чтобы закрепить Гит на файл

```
git init
```
получить информацию от гита о его текущем состаянии

```
git status
```
## Добавление коммита 

Добавить файл к следущему коммиту 

```
git add <Название файла.расширение>
```
после можно коментировать

```
git commit -m "коментарий"
```
